# Radix Converter

## なぜ作成したのか

- Reactの学習の中で、Reactの基礎を身に付けたかった為
- Next.jsのチュートリアルを修了し、Next.jsを使用したアプリケーションを作成したかった為
- TypeScriptを使用したアプリケーションを作成したかった為
- Tailwind CSSを使用してcssを適用し、Tailwind CSSの学習をしたかった為

## 内容

- 2進数,8進数,10進数,16進数の相互変換を実装
